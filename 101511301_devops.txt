COMP 3104 - DevOps Course Information

Course Overview:
COMP 3104 DevOps is a comprehensive course that bridges the gap between software development and IT operations. This course teaches students the principles, practices, and tools essential for modern software development and deployment.

Course Objectives:
- Understand DevOps culture and methodology
- Learn version control systems (Git and GitHub)
- Master continuous integration and continuous deployment (CI/CD)
- Gain hands-on experience with automation tools
- Learn containerization with Docker
- Understand infrastructure as code
- Practice collaborative development workflows

Key Topics Covered:
1. Version Control Systems
   - Git fundamentals and advanced features
   - GitHub workflows and collaboration
   - Branching strategies and pull requests

2. Continuous Integration/Continuous Deployment
   - GitHub Actions
   - Automated testing
   - Build automation
   - Deployment pipelines

3. Containerization and Orchestration
   - Docker containers
   - Container management
   - Microservices architecture

4. Infrastructure as Code
   - Configuration management
   - Infrastructure automation
   - Cloud platforms integration

Skills Developed:
- Problem-solving in development and operations
- Team collaboration and communication
- Automation mindset
- Modern software delivery practices
- Cloud-native application development

This course prepares students for real-world DevOps roles by providing practical experience with industry-standard tools and workflows. Students work on projects that simulate actual DevOps environments, learning to automate, monitor, and optimize software development lifecycles.
